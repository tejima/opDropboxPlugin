opDropboxPlugin

opDropboxPluginはOpenPNEにファイルアップロード機能を追加する、プラグインです。 Dropbox格納方式とDB格納方式が選択できます。

主な機能

・ファイルアップロード ・ファイルリスト ・日記貼り付け ・ファイル削除 ・管理画面ファイル管理

インストール

$ cd OpenPNE
$ symfony opPlugin install opDropboxPlugin

設定

管理画面にアクセスし、Dropboxモード、DBモードを選択します。デフォルトはDBモードです。 Dropboxモードを選択した場合は、DropboxのAPIキーが必要です。
